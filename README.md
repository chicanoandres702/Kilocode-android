# Kilo Code Monorepo

Welcome to the Kilo Code repository. This monorepo contains both the web-based AI assistant interface and the native Android client.

## Repository Structure

- `kilocode/`: Kilo Code server and core AI logic.
- `kilocode-android/`: Native Android client for Kilo Code.
- `src/`: Web-based Next.js interface.

## Getting Started

### Web Interface
1. Install dependencies: `bun install`
2. Start dev server: `bun dev`

### Android Client
See `kilocode-android/README.md` for instructions on building and running the Android application.
